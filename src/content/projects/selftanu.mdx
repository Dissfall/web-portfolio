## Project Overview

A comprehensive full-stack platform combining a modern React frontend with a robust Laravel backend, deployed on AWS infrastructure.

## Technical Highlights

- Next.js and React frontend for dynamic user interface
- Laravel backend providing RESTful API
- AWS Beanstalk for scalable backend deployment
- AWS WAF for security and DDoS protection
- AWS Amplify for frontend hosting and CI/CD
